14 # We create a simple subclass of c_char_p here because when the response
15 # type is set to c_char_p, you get a _Python_ string and there's no way
16 # to access the string's address inside the error checking function.
17 # In other words, you can't free the memory allocated inside GEOS. Previously,
18 # the return type would just be omitted and the integer address would be
19 # used -- but this allows us to be specific in the function definition and
20 # keeps the reference so it may be free'd.
21 class geos_char_p(c_char_p):
22 pass
