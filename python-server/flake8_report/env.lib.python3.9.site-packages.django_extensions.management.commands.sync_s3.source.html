3 Sync Media to S3
4 ================
5  
6 Django command that scans all files in your settings.MEDIA_ROOT and
7 settings.STATIC_ROOT folders and uploads them to S3 with the same directory
8 structure.
9  
10 This command can optionally do the following but it is off by default:
11 * gzip compress any CSS and Javascript files it finds and adds the appropriate
12 'Content-Encoding' header.
13 * set a far future 'Expires' header for optimal caching.
14 * upload only media or static files.
15 * use any other provider compatible with Amazon S3.
16 * set other than 'public-read' ACL.
17  
18 Note: This script requires the Python boto library and valid Amazon Web
19 Services API keys.
20  
21 Required settings.py variables:
22 AWS_ACCESS_KEY_ID = ''
23 AWS_SECRET_ACCESS_KEY = ''
24 AWS_BUCKET_NAME = ''
25  
26 When you call this command with the `--renamegzip` param, it will add
27 the '.gz' extension to the file name. But Safari just doesn't recognize
28 '.gz' files and your site won't work on it! To fix this problem, you can
29 set any other extension (like .jgz) in the `SYNC_S3_RENAME_GZIP_EXT`
30 variable.
31  
32 Command options are:
33 -p PREFIX, --prefix=PREFIX
34 The prefix to prepend to the path on S3.
35 --gzip Enables gzipping CSS and Javascript files.
36 --expires Enables setting a far future expires header.
37 --force Skip the file mtime check to force upload of all
38 files.
39 --filter-list Override default directory and file exclusion
40 filters. (enter as comma separated line)
41 --renamegzip Enables renaming of gzipped files by appending '.gz'.
42 to the original file name. This way your original
43 assets will not be replaced by the gzipped ones.
44 You can change the extension setting the
45 `SYNC_S3_RENAME_GZIP_EXT` var in your settings.py
46 file.
47 --invalidate Invalidates the objects in CloudFront after uploading
48 stuff to s3.
49 --media-only Only MEDIA_ROOT files will be uploaded to S3.
50 --static-only Only STATIC_ROOT files will be uploaded to S3.
51 --s3host Override default s3 host.
52 --acl Override default ACL settings ('public-read' if
53 settings.AWS_DEFAULT_ACL is not defined).
54  
55 TODO:
56 * Use fnmatch (or regex) to allow more complex FILTER_LIST rules.
57
